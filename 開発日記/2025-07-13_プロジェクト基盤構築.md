## 作成したファイル構造
### ディレクトリ構造
```
src/
├── app/
│   ├── cast/[id]/          # キャスト詳細画面
│   ├── menu/
│   │   ├── pricing/        # 料金画面
│   │   └── drinks/         # ドリンクメニュー画面
│   ├── admin/              # 管理画面
│   │   └── login/          # ログイン画面
│   └── api/                # API Routes
│       ├── casts/
│       ├── auth/
│       └── admin/
├── components/
│   ├── ui/                 # 基本UIコンポーネント
│   ├── cast/               # キャスト関連コンポーネント
│   ├── chart/              # チャートコンポーネント
│   ├── layout/             # レイアウトコンポーネント
│   └── admin/              # 管理画面コンポーネント
├── types/                  # TypeScript型定義
└── lib/                    # ユーティリティ
```

### 作成したコンポーネント
- **Button**: 基本ボタンコンポーネント（4種のバリアント）
- **Badge**: バッジ表示コンポーネント（動的カラー対応）
- **CastCard**: キャストカード（4:3比率、バッジ・統計表示対応）
- **CastGrid**: レスポンシブグリッド（縦画面3列、横画面5列）
- **RadarChart**: 6角形レーダーチャート（1-100スケール）
- **Navigation**: タブレット向けナビゲーション

### データベーススキーマ
- **casts**: キャスト基本情報
- **cast_photos**: 複数写真対応
- **cast_stats**: レーダーチャート用能力値
- **badges**: バッジマスタ
- **cast_badges**: キャストとバッジの関連
- **users**: 管理者ユーザー
