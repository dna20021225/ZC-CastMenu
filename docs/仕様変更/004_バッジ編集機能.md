### 仕様

- 管理画面に `/admin/badges` を新設
  - バッジ一覧表示（並び順は display_order）
  - 新規作成: 名前 + カラーピッカー
  - 既存編集: インライン編集（名前・色）
  - 削除: 関連する cast_badges もカスケード削除
  - ドラッグ＆ドロップで並び順変更（007 と同じ @dnd-kit）
- No.1 / No.2 / No.3 の排他処理（1人のみ付与）は廃止
  - すべてのバッジは複数キャストに自由に付与可能
  - 「No.1 を一番上に出したい」など順序操作は管理画面の display_order（007）で
- 管理ダッシュボードにバッジカード追加（4カード化）

### 変更内容

1. **API**
   - `POST /api/badges` 新規作成
   - `PUT /api/badges/[id]` 編集
   - `DELETE /api/badges/[id]` 削除
   - `PATCH /api/badges/reorder` 並び順保存
2. **キャスト更新API（PUT /api/casts/[id]）**
   - No.1/2/3 の排他削除ロジックを撤廃
3. **管理画面**
   - `/admin/badges` 新規作成（一覧 + 作成 + インライン編集 + 削除 + ドラッグ並び替え）
   - `/admin` のダッシュボードにバッジカード追加
