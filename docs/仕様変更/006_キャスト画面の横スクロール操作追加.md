### 仕様

- キャスト詳細画面 `/cast/[id]` で画面全体の横スワイプを検知
  - 左スワイプ → 次のキャスト
  - 右スワイプ → 前のキャスト
- 並び順は 007 で実装した `display_order` に従う（ホーム画面と同順）
- 端でループする（最後 → 最初 / 最初 → 最後）
- 写真エリア（`data-photo-carousel="true"`）内のスワイプは写真カルーセル優先のため無視

### 変更内容

1. **新規カスタムフック** `src/hooks/useHorizontalSwipe.ts`
   - TouchEvent ベースの横スワイプ検知
   - 縦移動が大きい場合（縦スクロール）は無視
   - `ignoreInsideSelector` で除外領域を指定可能
2. **キャスト詳細ページ** `src/app/cast/[id]/page.tsx`
   - 一覧API（`/api/casts?limit=100`）を別途取得し、display_order 順の ID 配列を保持
   - スワイプフックで前後キャストへ `router.push`
   - 写真エリアに `data-photo-carousel="true"` 属性を付与
